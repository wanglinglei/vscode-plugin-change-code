## 代码格式转换

### description
通过插件快速对选中代码的格式转换,目前可转换格式: 中划线/下划线/大写常量/驼峰.
![编辑器示例](https://mdn.alipayobjects.com/huamei_vwpth6/afts/img/A*0-42RbqZ6t8AAAAAAAAAAAAADmZ7AQ/original)
### 转换示例

|原始代码|转换格式|转换结果|
|:------:   |:------:|:------:|
|aaaBbb|中划线|aaa-bbb|
|aaaBbb|下划线|aaa_bbb|
|aaaBbb|大写常量|AAA_BBB|
|aaa bbb|驼峰|aaaBbb|
|aaa bbb|中划线|aaa-bbb|
|aaa bbb|下划线|aaa_bbb|
|aaa bbb|大写常量|AAA_BBB|
